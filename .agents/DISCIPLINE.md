# 紀律操作協議（Agent 輔助 × 你執行）

目標：你不靠當下感覺做決定；Agent 依規則產出「今日只做清單」，你只在約定時段執行。

## 分工

| 角色 | 負責 | 不負責 |
|------|------|--------|
| **Agent** | 抓報價／對停損／對 Level／產出清單與優先序 | 不下單、不保證獲利 |
| **你** | 在時段內依清單下單、回報成交結果 | 不盤中發明新理由、不攤平、不追價 |

## 每日節奏（平日）

1. **開盤前／早盤（可問 Agent）**：只要「今日清單」，不要討論感覺。
2. **09:00–09:30**：凍結，不市價砍倉。
3. **10:00–11:30**：只執行已在**收盤確認**後的減碼／出清（反彈波）；不要因盤中推播殺多單。
4. **約 13:10**：系統做**收盤確認破防守**（個股停損只在這時推）。
5. **13:40 後**：依清單預掛／隔日開盤執行。
6. **晚上（可選）**：美股／外匯／加密觀測。

> 偵測輔助不跟外資盤中做妖：個股破防守**不盤中推播**。反1／出清標的逢高提醒仍可保留。

## 賣出只有三種（聽起來才不會像只會賠）

| 類型 | 何時 | 意義 |
|------|------|------|
| **達標停利** | 相對成本約 **+12%** 先賣 ~1/3；**+25%** 再賣 ~1/3 | 把賺到的錢落袋 |
| **移動停利** | 收盤跌破 **5MA／10MA** | 趨勢轉弱，鎖定剩下漲幅（不是砍虧） |
| **停損** | 收盤跌破 **5日低** | 真的認錯出場 |

長抱標的（0050／正2／VOO）：主要靠移動停利＋年線，不強迫到點全出，避免賣飛牛市。

## 決策優先序（固定）

1. `force_exit` 出清窗（00632R、00882）— 大跌日優先賣反1  
2. 已成立「兩日不站回」的內資股 — 出清剩餘  
3. 外資主導股破防守 — 至少減半（2301）  
4. 其餘內資股 — 收盤確認／兩日法則，不盤中情緒砍  
5. 正2 — Level 3 **不加碼**；年線未破可續抱既有部位  
6. 出清資金 — Level 3 先抱現金，不急著買回

## 暫時焦點（大輸後修復期）

- **現金開關**：總現金約 600 萬，只開放 **100 萬**
- **核准新買**：`0050`、`00631L`（正2；Level≥3 不加）、台銀黃金、`VOO`+`VXUS`（7:3）、成長袖 `QQQ`/`QQQM`、既有 USDT 活期
- **成長同等風險**：QQQ ⇄ QQQM ⇄ 00631L（進攻總額控管）
- **只抱不加**：美債、BTC/ETH、剩餘個股
- **剔除**：原油／原物料 ETF、山寨、新開個股
- **通知**：雲端 GitHub Actions + 本機 `run_all_alerts.py`

## 禁止清單

- 開盤殺盤市價出清（除非破產級流動性危機）
- Level 3 加碼正2／攤平個股
- 「基本面好」推翻已觸發的停損
- 一天內反覆改規則

## 你回報給 Agent 的格式（成交後貼上即可）

```
成交：
- 代號 / 張數 / 價格 / 剩餘張數
待確認：
- …
```

Agent 會據此更新 `config/my_targets.json` 建議與明日清單。
